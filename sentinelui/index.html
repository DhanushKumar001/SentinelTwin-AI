ℹ️ How Scenarios Work

When a scenario is triggered, sensor values change dynamically. The AI modules detect the changes and respond:

1. Sensor Injection
Fault multipliers applied to machine sensors
2. Anomaly Detection
3-detector ensemble confirms deviation
3. Predictive Alert
TCN/LSTM/TFT raise failure probability
4. Self-Healing
RL controller applies corrective actions
5. 3D Twin Update
Machine shakes, sparks, warning lights activate
6. Auto Resolution
Self-healing restores normal state in ~30s
